# 🏴‍☠️ Manga Reader

Легковесная веб-читалка для манги в формате PDF, написанная на Python с использованием библиотеки **Streamlit**. Идеально подходит для чтения манги с ПК и мобильных устройств, используя файлы прямо из облачных хранилищ (OneDrive, Google Drive), синхронизированных с компьютером.

## ✨ Основные возможности

* 📱 **Оптимизация под мобильные устройства**: Страницы нарезаются на удобные "части" (по 5 страниц), чтобы браузер не зависал при загрузке 200-страничных томов.
* 💾 **Кэширование**: Изображения кэшируются в памяти, что обеспечивает моментальное перелистывание страниц вперед и назад.
* 📍 **Умные закладки**: Приложение запоминает:
  * Текущий читаемый том.
  * Номер части (главы).
  * Точную картинку (с помощью кнопки "Запомнить место").
* 🔄 **Синхронизация (Progress State)**: Прогресс чтения сохраняется в локальный `read_progress.json`. Вы можете читать на ПК, переключиться на телефон, обновить страницу — и продолжить с того же места!
* 🔍 **Настраиваемый масштаб**: Ползунок ширины страницы для комфортного чтения на широких мониторах (значение масштаба также сохраняется между сессиями).
* ⬇️ **Плавный авто-скролл**: При переходе на новую часть или загрузке закладки страница автоматически прокручивается к нужному месту.

## 🛠 Установка и запуск

1. **Клонируйте репозиторий** и перейдите в папку проекта.
2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
   *(Основные библиотеки: `streamlit`, `PyMuPDF`)*
3. **Настройте путь к вашей манге**:
   Откройте `app.py` и измените переменную `MANGA_FOLDER` на путь к вашей папке с PDF файлами (например, к папке OneDrive).
   ```python
   MANGA_FOLDER = r"C:\Users\...\OneDrive\Manga"
   ```
4. **Запустите приложение**:
   ```bash
   streamlit run app.py
   ```

## 🌍 Доступ с мобильного телефона
Если вы используете VS Code, перейдите во вкладку **Ports** (Порты) в нижней панели, пробросьте порт `8501` и сделайте его **Public**. Скопируйте ссылку и откройте её на смартфоне! Компьютер при этом должен оставаться включенным.

## ⚙️ Структура проекта
* `app.py` — Главный файл с логикой приложения и веб-интерфейсом.
* `requirements.txt` — Список зависимостей Python.
* `read_progress.json` — Автоматически генерируемый файл, хранящий ваш прогресс чтения.
* `.gitignore` — Исключения для Git-репозитория.
